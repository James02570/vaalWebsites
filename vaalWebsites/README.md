# Vaal Websites Frontend

This folder is ready to deploy on Vercel as a static site.

## Deploy on Vercel

1. Set the Vercel project root to this `frontend` folder.
2. Deploy with no build step.
3. If your backend is hosted somewhere other than `http://localhost:3000`, update `config.js` with the live API base URL before deploying.

## Local Preview

Open `index.html` with a static file server, or run any simple local server from this folder.
